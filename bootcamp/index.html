  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta element contains metadata.
  • The title element contains the title and is important for search engine optimization.
  • The style element defines the CSS styles associated with the site.
  • The base element defines the base URL for a webpage.
  • The link element contains an external resource to the HTML document.
  • The script element embeds client-side scripts in the HTML document
  • The heading elements create a series of headings in descending importance from h1 to h6.
  • The paragraph element creates an area of text.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • You need to include a link to your stylesheet in your HTML.
  • Using * as a selector essentially selects everything
  • By using a class selector, you can simultaneously style everything of that class.
  • To style an element, you must specify the selector, the property and the value.
